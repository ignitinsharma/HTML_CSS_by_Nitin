b tag

The b element is used to draw the reader's attention to the element's contents, which are not otherwise granted special importance.

i tag

The i element represents a range of text that is set off from the normal text for some reason, such as idiomatic text, technical terms, taxonomical designations, among others. The content inside is typically displayed in italic.

u tag

The u element represents a span of inline text which should be rendered in a way that indicates that it has a not-textual annotation. It rendered as an solid underlined text, but it can be changed using CSS properties.

select tag

The select element represents a control that provides a menu of options.

button tag

The button element is an interactive element activated by a user with a mouse, keyboard, finger, voice command, or other assistive technology. Once activated, it then performs a programmable action, such as submitting a form or opening a dialog.

by default, HTML buttons are presented in a style resembling the platform the user agent runs on, but you can change buttons appearance with CSS.

hr tag

The hr element represents a thematic break between paragraph-level elements.

br tag

The br element produces a line break in text (carriage-return).
